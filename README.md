## ~/.config

The `config/` directory holds backups of `~/.config` app configs. Each app
directory is symlinked back into `~/.config`, so edits land in this repo
automatically:

```sh
ln -s ~/projects/dotfiles/config/<app> ~/.config/<app>
```

Backed up: `hypr`, `waybar`, `wallust`, `rofi`, `ghostty`, `btop`, `cava`,
`swaync`, `swappy`, `nwg-displays`, `waybar-weather`.

Intentionally excluded (see `.gitignore` / not linked):
- `config/waybar-weather/{geolocation,cityname}` — private location data
- `config/hypr/monitors.conf` — machine-specific, auto-generated by nwg-displays
- `zsh/plugins`, `mise`, `xfce4`, `opencode` — vendored deps / runtime state /
  linked elsewhere

Note: `hypr/*.conf` hardcode `/home/nyx/...` paths; adjust on restore if the
username differs.
